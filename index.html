The flex-shrink Property

flex-shrink is a CSS property that controls how much a flex item will shrink when there is not enough space available in the flex container. The flex-shrink property takes a unitless value that represents the proportion by which the flex item should shrink relative to other flex items in the container. For example, if one flex item has a flex-shrink value of 1 and another has a value of 2, the second item will shrink twice as much as the first when there is not enough space. If a flex item has a flex-shrink value of 0, it will not shrink at all.

The Align-self Property

align-self is a CSS property that is used to control the vertical alignment of a single flex item within a flex container. Normally, the align-items property is used to align all flex items within a container. However, sometimes you may want to align a single item differently than all the others. In this case, you can use the align-self property on that particular item to override the align-items value for that item only
